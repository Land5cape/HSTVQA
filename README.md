# HSTVQA
## Test Demo
The model weights provided in `model-save/XXX/HSTVQA.pth` are the saved weights when running on the KoNViD-1k, CVD2014, and LIVE-VQC, respectively.

Please run  
```
python example.py
```  
to start the demo.

```
pip install -r requirements.txt
```

- TensorboartX 2.1
